V. Practical Considerations & Implementation

1. Choosing the Right Test

  • Determine data type: continuous, categorical, ordinal
  • Check distribution: normality vs non-normality
  • Sample size: Small sample → nonparametric methods may be safer
  • Paired vs independent samples
  • Purpose: Compare means, proportions, variances, or relationships

2. Effect Sizes

Effect size quantifies the magnitude of a phenomenon, independent of sample size.

  • Cohen's d: 
    d = (M₁ - M₂) / SDpooled
  • Correlation coefficient: 
    r = cov(X,Y) / (σX * σY)
  • Odds ratio: 
    OR = (a/b)/(c/d)
  • Eta-squared for ANOVA: 
    η² = SSbetween / SStotal

3. Multiple Testing & Adjustments

  • Bonferroni correction: αadjusted = α / number of tests
  • Holm-Bonferroni: Stepwise and less conservative
  • False Discovery Rate (FDR) – Benjamini-Hochberg method

4. Sample Size & Power Analysis

Determines required sample size to detect effect with confidence.

  • Power (1-β): Probability to detect true effect
  • Depends on α, effect size, variability, and sample size
  • Tools: G*Power, R packages (pwr)

5. Assumptions & Diagnostics

Normality Checks

  • Shapiro-Wilk test
  • Kolmogorov-Smirnov test
  • QQ-plots & histograms

Homogeneity of Variance

  • Levene's test
  • Brown-Forsythe test

Independence

  • Check study design & randomization

6. Implementation Tips

  • Always visualize data first (boxplots, histograms, scatterplots)
  • Use robust methods if assumptions fail
  • Report both p-values and effect sizes
  • Document every step for reproducibility
