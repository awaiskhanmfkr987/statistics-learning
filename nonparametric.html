III. Nonparametric Tests: Comprehensive Theory

1. Overview of Nonparametric Methods

Nonparametric tests are used when data do not satisfy parametric assumptions (normality, equal variance, interval scale). They often use ranks instead of raw data and are robust to outliers.

2. Common Nonparametric Tests

A. Sign Test

Tests median of a single sample against a hypothesized value or compares paired samples. Only the direction of difference is considered.

n₊ = number of positive differences
n₋ = number of negative differences

B. Wilcoxon Signed-Rank Test

Tests median differences of paired data considering ranks and signs.

W = sum of ranks for positive differences

C. Mann–Whitney U Test

Compares two independent samples to check if they come from the same distribution.

U₁ = n₁*n₂ + (n₁*(n₁+1))/2 - R₁
U₂ = n₁*n₂ + (n₂*(n₂+1))/2 - R₂

Where R₁ and R₂ are sums of ranks for each group.

D. Kruskal–Wallis Test

Nonparametric alternative to one-way ANOVA for >2 groups.

H = (12 / (N*(N+1))) * Σ(Ri²/ni) - 3*(N+1)

Where Ri is sum of ranks for group i, ni is sample size of group i, N is total sample size.

E. Friedman Test

Nonparametric equivalent of repeated measures ANOVA.

χ²_F = (12 / (n*k*(k+1))) * Σ(Rj²) - 3*n*(k+1)

Where Rj is sum of ranks in block j, n = number of blocks, k = number of treatments.

3. Chi-Square Test

Tests independence or goodness-of-fit for categorical data.

χ² = Σ((O - E)² / E)

Where O = observed frequency, E = expected frequency.

4. Advantages of Nonparametric Tests

  • No strict assumption of normality.
  • Suitable for ordinal or ranked data.
  • Robust to outliers.

5. Limitations

  • Less powerful than parametric tests if parametric assumptions are met.
  • Cannot estimate parameters like mean and variance directly.
