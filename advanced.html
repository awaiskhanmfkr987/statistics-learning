IV. Advanced and Specialized Tests

1. Bayesian Hypothesis Testing

Uses prior information and observed data to update beliefs about hypotheses.

Bayes Factor: BF₁₀ = P(data|H₁)/P(data|H₀)

Interpretation:

  • BF > 1: Evidence favors H₁
  • BF < 1: Evidence favors H₀
  • BF ≈ 1: Inconclusive

Posterior Probability:

P(H₁|data) = [BF₁₀ × prior odds] / [1 + BF₁₀ × prior odds]

2. Resampling Methods

A. Bootstrap

Generates repeated samples to estimate the sampling distribution of a statistic.

Nonparametric: Resample with replacement from data
Parametric: Resample from fitted model
p-value = proportion of bootstrap statistics ≥ observed

B. Permutation Tests

Tests exact null distribution by permuting labels.

Exact permutation: enumerate all permutations
Monte Carlo: sample random permutations
Assumes exchangeability under H₀

3. Time Series Tests

Stationarity Tests

  • Augmented Dickey-Fuller (ADF): H₀ = unit root (non-stationary)
  • KPSS: H₀ = stationary
  • Phillips-Perron: Nonparametric correction for autocorrelation

Cointegration Tests

  • Engle-Granger two-step
  • Johansen VAR-based test

Causality Tests

  • Granger causality: F-test in VAR
  • Transfer entropy: Information-theoretic measure

4. Survival Analysis Tests

Log-Rank Test

χ² = Σ((Oᵢ - Eᵢ)² / Eᵢ)

Wilcoxon-Breslow-Gehan

Weights by risk set size

Cox Proportional Hazards

h(t|x) = h₀(t) * exp(β'x)

Accelerated Failure Time Models

log T = β'x + σε

5. Multivariate Tests

Hotelling's T²

T² = n * (x̄ - μ₀)' S⁻¹ (x̄ - μ₀)

MANOVA

Wilks' Lambda: Λ = |E| / |E + H|
Pillai's Trace: V = trace[H(H+E)⁻¹]
Hotelling-Lawley Trace: U = trace[E⁻¹H]
Roy's Largest Root: θ = largest eigenvalue of E⁻¹H

Discriminant Analysis

Maximizes ratio of between-group variance to within-group variance.

6. Robust Statistical Tests

  • M-estimators: Minimize Σρ(xᵢ, θ)
  • Influence Function: Measures sensitivity to contamination
  • Trimmed/Winsorized t-tests: Reduce effect of outliers
  • R-estimators: Based on ranks
