I. Foundational Theoretical Framework

1. Hypothesis Testing Philosophy

Statistical testing represents the formal process of making inferences about populations from samples...

  • Null Hypothesis (H₀): The default or skeptical position.
  • Alternative Hypothesis (H₁): The research claim representing an effect, difference, or relationship.

2. Decision Theory Framework

Every statistical test embodies an optimal decision rule derived from statistical decision theory...

3. Error Taxonomy

  • Type I Error (α): Rejecting a true null hypothesis
  • Type II Error (β): Failing to reject a false null hypothesis
  • Power (1-β): Probability of correctly rejecting a false null
  • Effect Size: Quantification of the magnitude of a phenomenon
