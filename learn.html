I. Introduction to Statistical Tests

Statistical tests are inferential tools used to draw conclusions about a population using sample data. They help researchers decide whether observed differences, relationships, or patterns are real or occurred by chance.

II. Parametric Tests

Definition: Parametric tests assume that data follows a specific distribution (usually normal) and that population parameters such as mean and variance exist.

Key Assumptions

  • Normality of data
  • Homogeneity of variance
  • Interval or ratio scale measurement
  • Independence of observations

3. Analysis of Variance (ANOVA)

Used to compare means of three or more groups.

  • One-Way ANOVA: One independent factor
  • Two-Way ANOVA: Two independent factors and interaction
  • Repeated Measures ANOVA: Same subjects measured repeatedly
  • MANOVA: Multiple dependent variables
  • ANCOVA: Controls covariates

4. Correlation & Regression

  • Pearson Correlation (r): Linear relationship between two continuous variables
  • Simple Linear Regression: Predicts outcome using one predictor
  • Multiple Regression: Uses multiple predictors
